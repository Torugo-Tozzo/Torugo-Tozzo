## Projetos

*(Clique no título de cada projeto para acessar o seu respectivo repositório)*

- 📱 [**Emater-App**](https://github.com/Torugo-Tozzo/Emater-App)
  Aplicativo mobile para Cálculo do Valor Relativo do Alimento (VRA), desenvolvido como meu Trabalho de Conclusão de Curso (TCC) na UNIFAL-MG. Feito em React Native com Expo e banco de dados local SQLite (funciona offline). Interface voltada ao produtor rural.
- 🏪 [**front-tozzo.uk**](https://github.com/Torugo-Tozzo/front-tozzo.uk)
  Frontend da plataforma Tozzo.uk (sistema de gestão para restaurantes e bares). Construído com React, TypeScript e Vite. Atualmente em produção e utilizado diariamente na nossa lanchonete e por outros comerciantes locais. [tozzo.uk](https://tozzo.uk)
- 🍔 [**TozzoBurger**](https://github.com/Torugo-Tozzo/TozzoBurger)
  Aplicativo de gestão de vendas e impressão direta de pedidos na cozinha para a lanchonete Tozzo Burger. Desenvolvido em React Native (Expo) conectando-se à API nativa do tozzo.uk.
- 🔍 [**Uaihoo**](https://github.com/Torugo-Tozzo/Uaihoo)
  Buscador de documentos desenvolvido como introdução ao Elastic Search, em colaboração com Lucas Justino. Feito com JS no frontend e Java no backend.
- 🖼️ [**Trabalho1_PDI**](https://github.com/Torugo-Tozzo/Trabalho1_PDI)
  Código em C para Processamento Digital de Imagens. Realiza a transformação de imagens em formato `.pgm` em versão texto (ASCII).
